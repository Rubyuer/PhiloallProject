# 乐菲足球

  "乐菲足球"是一款线上预约报名赛事、预约足球场的应用。
  
  a.预约球场
  "乐菲足球"通过户内外进行分类足球场;
  通过预约日期和预约时间点,预约人信息等提交预约信息,生成对应场馆的预约订单。
  
  b.报名赛事
  "赛事"提供成都范围内的足球组队赛事;
  通过输入联系人方式和电话，以及报名人的身份证号码进行报名(身份证号码将用于报名的第二凭证)。
  
  c.资讯列表
  "资讯"所列出全网的关于足球的新闻和赛事通知消息。
  
  d.预约列表
  用户预约的足球场将在预约订单栏目展示，包含(已预约、已取消、已完成等类型)。
  
  e.我的
  "我的"显示用户基本数据信息:头像、昵称等;
  我的信息、我的报名、修改登录密码等功能。
